# BrowserGrabber

grabs saved passwords from basically every browser on windows and sends them to a discord webhook

## what browsers it supports

**chromium (chrome engine)**
- chrome, chrome beta, chrome dev
- edge, edge beta
- brave, brave beta
- opera, opera gx
- vivaldi
- yandex browser
- chromium
- epic privacy browser
- comodo dragon
- centbrowser
- torch, slimjet, orbitum, amigo, chedot, uc browser, coccoc, iridium, 7star, sputnik

**firefox engine**
- firefox
- waterfox
- pale moon
- seamonkey
- thunderbird

## what gets sent to discord
- pc username
- hostname
- ip address
- every saved password (url, username, password) from all browsers found :)
